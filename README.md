# douban-movie
## 简介

- `webpack` + `vue` + `vuex` + `vue-router` + `iView`仿写豆瓣wap端。
- `rem响应式布局` + `Grid` + `Flexbox`的布局方式。
- 使用`http-middleware-proxy`实现跨域。
- 提取公共组件。

使用vue全家桶的应用。初步设想是移动端，响应式。

[豆瓣api](https://developers.douban.com/wiki/?title=movie_v2)
